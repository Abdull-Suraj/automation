# Zynapse AI — Brand Website

A modern, responsive brand website for **Zynapse AI**, an AI automation agency that provides intelligent business solutions.

## 🚀 Features

- **Island-style floating navbar** — Modern navigation bar with glass morphism effect
- **Fully responsive** — Optimized for all screen sizes (mobile, tablet, desktop)
- **Smooth scroll animations** — Elements animate into view as you scroll
- **Modern dark theme** — Premium dark design with gradient accents
- **Animated statistics counter** — Numbers count up when scrolled into view
- **Interactive form** — Contact form with submission feedback

## 📁 Project Structure

```
├── index.html       # Main HTML page
├── css/
│   └── styles.css   # All styles (variables, layout, responsive)
├── js/
│   └── main.js      # Navbar, animations, form handling
└── README.md
```

## 💻 Getting Started

Simply open `index.html` in any modern web browser. No build tools required.

```bash
# Or serve locally
npx serve .
```

- **Font**: Inter + Space Grotesk (Google Fonts)
